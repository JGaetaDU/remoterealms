Remote Realms is a browser-based RPG focused on skill training, gathering, crafting, and long-term progression, and this calculator is designed to make farming and XP planning easier while you play.

Community Notes and Development Process

This section documents how the calculator evolved through spreadsheet testing, player review, and website revisions.

Community testing

  • The calculator began as a spreadsheet used to test farming XP progression and crop efficiency.
  • Players reviewed early calculations and helped confirm crop values, seeds required, and interface clarity.
  • Player feedback led to replacing the original XP input structure with a simpler total XP model that matches what players see in game.
  • Multiple versions of the tool were shared so players could compare results against their own in-game progress.

Design decisions

  • The XP calculator now uses current total XP instead of XP within a level range.
  • Crop sorting includes level unlock, alphabetical order, patch grouping, XP value, and growth-time views.
  • Dataset tables are visible as tabs so players can inspect the values behind the calculator.
  • Harvest assumptions are shown directly in the farming calculator to make the results easier to interpret.

Current limitations

  • Some crops still have incomplete XP values while testing continues.
  • Compost and other modifiers are intentionally excluded until consistent values can be verified.
  • The calculator is built for planning and comparison, not full simulation of every farming mechanic.

Ongoing development

  • The tool will continue to evolve as players test new mechanics and provide feedback.
  • Future versions may separate the datasets into external files for easier updating when the game changes.
  • Additional features can be added after the academic project is complete and the current core resource is stable.
