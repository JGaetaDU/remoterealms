How the XP math works

Term Meaning
Start Level Total XP Total cumulative XP required to be at your selected current level.
Current XP Within Current Level Additional XP earned after reaching the selected current level.
Remaining XP XP still needed after subtracting your current within-level progress.
XP Needed For Full Goal Total XP gap between the selected current level baseline and the target level baseline.
Suggested future tweak: you can later add a dropdown for different skills if Remote Realms ever uses skill-specific XP tables.

Data Notes and Assumptions

This page helps make the calculator more transparent for players.

Current scope

  • Includes XP progression data for levels 1–99.
  • Includes farming crop data used for planning and comparison.
  • Excludes compost, watering bonuses, and other mechanics that are still under testing.

Community-requested changes included in this version

  • Crop sorting by level unlock.
  • Clearer explanation of how XP is calculated.
  • Seeds needed estimate.
  • Prevention of negative or impossible inputs.
  • Improved visibility and layout.

Privacy and community contribution

If you later publish this site, use aggregated feedback or anonymous notes rather than identifiable player names unless you have permission.

How to replace the sample data with your real data

  1. Export your XP_Data and Farming_Data sheets to CSV or JSON.
  2. Replace the arrays in the <script> section at the bottom of this file.
  3. Upload this file to GitHub Pages or split it into separate HTML, CSS, and JS files later.
